# Unity Finite State Machine 
This is the miHoYo 2026 Technical Artist (TA) Assessment Project.<br>
Subsequently, I used it to practice character state machines, IK algorithms for related animations, and procedural animation.<br>

The State Machine utilizes `PlayerState` as its base class; each state operates independently, and state transitions are managed via the `StateMachine`, thereby reducing overhead within the Unity Animation System.<br>

Control:<br>
AWSD----movement<br>
Ctrl----change walk and run<br>
Shift----Sprint<br>
Left Mouse----Attack<br>
R----Aim
